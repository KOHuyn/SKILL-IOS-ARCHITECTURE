# iOS Architecture Skill

Claude Code skill for iOS Clean Architecture with MVI/MVVM patterns.

## Installation

Clone to your project root:
```bash
git clone https://github.com/KOHuyn/SKILL-IOS-ARCHITECTURE.git .
```

Or add as submodule:
```bash
git submodule add https://github.com/KOHuyn/SKILL-IOS-ARCHITECTURE.git
```

## What's Included

- **Clean Architecture** - Domain/Data/Presentation layers
- **MVI Pattern** - Unidirectional state with Store, Intent, Reducer
- **MVVM Pattern** - ViewModel with UiState
- **Testing Patterns** - Unit testing strategies
